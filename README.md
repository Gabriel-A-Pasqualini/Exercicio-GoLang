# Website Monitor (Go)  

A simple Go project created as part of a course. This program monitors the availability of websites, logs their status, and provides a menu-based interface for user interaction.

## Features

- Monitor multiple websites by reading URLs from `sites.txt`
- Log website status (online/offline) with timestamps in `log.txt`
- Multiple monitoring rounds with a configurable delay
- Menu interface to start monitoring, view logs, and exit
